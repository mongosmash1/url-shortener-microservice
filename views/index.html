freeCodeCamp Backend API Projects - URL Shortener Microservice

Simple API to return a shortened URL that redirects to a requested website

Example URL creation usage:
https://ms1-url-micro.glitch.me/api/url/https://www.google.com
https://ms1-url-micro.glitch.me/api/url/http://foo.com:80

Example URL creation output:
{ "original_url":"http://foo.com:80", "short_url":"https://ms1-url-micro.glitch.me/ff6c4a0b" }


Example URL usage:
https://ms1-url-micro.glitch.me/290b2b60

Example URL redirects to:
https://www.google.com
